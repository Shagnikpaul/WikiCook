### PSQL Connect to database

`psql -U your_user -d postgres`






## List databases
`SELECT datname FROM pg_database;`

in command line just type 
`\l`


## To create database
`create database test;`


## connect to a database 
`\c test`


## drop a database

`drop database test`

## create a table

`create <tablename> (<variables>, )`

## table descritpion
`\d <tablename>`
